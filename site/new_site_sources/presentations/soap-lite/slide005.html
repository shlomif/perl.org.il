What is SOAP?

  • SOAP stands for Simple Object Access Protocol, and is one of the latest in a long line of projects like CORBA, DCOM, and XML-RPC.

  • Lets you make remote procedure calls (RPC) to a server through common network protocols like HTTP and SMTP through XML messages.

  • I wrote a client to access a database through a web based server.
