Talking to a Non-Perl Support Person

  • Many technical support people don't know Perl

  • Many SOAP server programmers write their servers in Java or C++.

  • This means that if your script is having problems, then sending the script to a support person will not help you.

  • Instead, send them the XML you are sending and the responses you see from their server.

  • This is how I was able to resolve several server side crashes.
