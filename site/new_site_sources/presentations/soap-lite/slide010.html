The WSDL to the Rescue

  • There is a method when creating the SOAP object called service().

  • You can supply a WSDL file, or an HTTP path to one.

  • This file contains a descriptor of all the possible methods and data structures for the SOAP service in XML format.

  • This file can contain the network location of the service's CGI interface.

  • SOAP::Lite can use this service descriptor file to build and evaluate SOAP messages.
