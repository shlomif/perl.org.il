What can be learned about an internet user

The current web technology enables to reveal an anonymous visitor:
  • geographical location (not perfectly)
  • time zone
  • spoken language
  • browser used
  • operating system used
  • whether a user searched for some keyword to come here
  • a time spent on a certain page (not perfectly)
  • the exact path of visited urls

More complex studies may include, for example, learning what users search for, what users searching for "motherboard" visit, what Swedish users usually buy, how many users that followed your advertisement actually purchase and so on.
