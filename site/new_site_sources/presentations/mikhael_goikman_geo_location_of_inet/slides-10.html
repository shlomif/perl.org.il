Perl modules dealing with geo locations

  • Geo::IP - convert host name or ip to country code or name
  • Locale::Country - ISO codes for country identification (ISO 3166)
  • Locale::SubCountry - convert state, province, county etc. names to/from code
  • Geography::States - map states and provinces to their codes
  • Geo::IPfree - similar to Geo::IP, but different database
