What a geek can do to preserve privacy

Mostly every user information listed previously can be blocked:

  • turning off javascript, java and other browser scripting/plugins
  • turning off referrers
  • turning off browser's User-Agent
  • use anonymous proxies

The last thing is not usually done, since it may be not that convenient. Then a user ip (and thus his geographical location) can be revealed.
