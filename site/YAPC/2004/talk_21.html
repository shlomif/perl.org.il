Title: Proper fields in Perl objects
Person: Gaal Yahas
Length: 15
Language: Hebrew
Abstract:
Perl OOP is very flexible, but rather hackish. One issue is how to implement member variables. This talk discusses the Class::Accessor approach, which is simple, elegant, and fast.
